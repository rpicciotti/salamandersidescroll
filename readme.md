## Prereqs

Deno: https://deno.com/manual/getting_started/installation

## Install

```
git clone git@github.com:timschwab/chesswar.git
cd chesswar
```

## Running the api server

```
deno task api
```

## Running the web server

```
deno task build
deno task web
```
